# Space-Impact
A copy of the Nokia 3310 game "Space Impact" made in C++ using tools provided by WinApi:
  - Direct2D for graphics
  - DirectWrite for font rendering
  - Windows Multimedia API for sound
  
# Build
Run GenerateProject.bat

# How to play
  - W: go up
  - S: go down
  - Space: shoot
